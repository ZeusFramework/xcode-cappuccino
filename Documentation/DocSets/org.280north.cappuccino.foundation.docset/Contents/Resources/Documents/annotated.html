Luokkalista

Tässä ovat luokat, tietueet ja yhdisteet lyhyen selitteen kanssa:
CPArrayA mutable array backed by a JavaScript Array.A mutable array class backed by a JavaScript Array. There is also a CPMutableArray class, but it is just a child class of this class with an empty implementation. All mutable functionality is implemented directly in CPArray
CPAttributedStringA mutable character string with attributes
CPBundleGroups information about an application's code & resources
CPCoderDefines methods for use when archiving & restoring (enc/decoding)
CPCountedSetAn mutable collection which may contain a specific object numerous times
CPDataA Cappuccino wrapper for any data type
CPDateA representation of a single point in time
CPDictionaryA mutable key-value pair collection
CPEnumeratorDefines an interface for enumerators
CPExceptionUsed to implement exception handling (creating & raising)
CPFunctionOperationRepresents an operation using a JavaScript function that can be run in an CPOperationQueue
CPHTTPURLResponse
CPIndexSetA collection of unique integers
CPInvocationAn object representation of a message
CPInvocationOperationRepresents an operation using an invocation that can be run in an CPOperationQueue
CPJSONPConnectionAllows cross domain connections using JSONP protocol
CPKeyedArchiverImplements keyed archiving of object graphs (e.g. for storing data)
CPKeyedUnarchiverUnarchives objects created using CPKeyedArchiver
CPMutableArray
CPMutableAttributedString
CPMutableDictionary
CPMutableIndexSet
CPMutableSet
CPNotificationA notification that can be posted to a CPNotificationCenter
CPNotificationCenterSends messages (CPNotification) between objects
CPNullAn object representation of nil
CPNumberA bridged object to native Javascript numbers
CPObjectThe root class from which most classes are subclassed
CPOperationRepresents an operation that can be run in an CPOperationQueue
CPOperationQueueRepresents an operation queue that can run CPOperations
CPPropertyListSerialization
CPProxy
CPRunLoopThe main run loop for the application
CPSetAn unordered collection of objects
CPSortDescriptorHolds attributes necessary to describe how to sort a set of objects
CPStringAn immutable string (collection of characters)
CPTimerA timer object that can send a message after the given time interval
CPUndoManagerA general mechanism for user action "undo"
CPURL
CPURLConnectionProvides loading of a URL request
CPURLRequestContains data obtained during a request made with CPURLConnection
CPURLResponseProtocol agnostic information about a request to a specifc URL
CPUserSessionManager
CPValueA generic "value". Can be subclassed to hold specific data types
CPWebDAVManager
